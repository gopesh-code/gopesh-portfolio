# Gopesh's Portfolio

## Overview

This is a personal portfolio website built to showcase my skills, projects, and achievements. The website is developed using **React.js** for the frontend and **Node.js** for backend sanity management. It includes sections for project showcases, skill highlights, and an "About Me" page.

## Features

- **Dynamic Portfolio**: Displays details about completed projects with images, descriptions, and links to GitHub repositories.
- **Responsive Design**: Fully responsive across devices, using SCSS for styling.
- **Backend Support**: Powered by a Node.js backend to fetch and display content dynamically.

## Technologies Used

- **React.js**: For building the frontend user interface.
- **Node.js**: Backend for handling data and content fetching.
- **SCSS**: For styling the frontend.
- **HTML/CSS**: Basic web structure and design components.

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/gopesh-code/gopesh-portfolio.git
   cd gopesh-portfolio
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm start
   ```
4. **Access the website: Open a browser and navigate to http://localhost:3000.**
